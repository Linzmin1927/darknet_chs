Darknet，一个纯C语言实现的深度学习框架，源自 https://github.com/pjreddie/darknet
Redmon等人在此框架下实现了目前最高水平的目标检测模型(Yolo V3)
本项目主要工作,在Darknet下实现: 
MobileNet(已完成)
MobileNet V2(Debuging)
ShuffleNet(阅读文献中)
ShuffleNet V2(阅读文献中)
Deep Compression(暂定)
