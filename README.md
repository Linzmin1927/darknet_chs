Darknet，一个纯C语言实现的深度学习框架，源自 https://github.com/pjreddie/darknet <br>
Redmon等人在此框架下实现了目前最高水平的目标检测模型(Yolo V3) <br>
本项目主要工作,在Darknet下实现:  <br>
MobileNet(已完成) <br>
MobileNet V2(Debuging) <br>
ShuffleNet(阅读文献中) <br>
ShuffleNet V2(阅读文献中) <br>
Deep Compression(暂定) <br>
